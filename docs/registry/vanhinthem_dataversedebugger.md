# DataverseDebugger

> Dataverse Debugger is an open-source developer tool for locally executing and debugging Microsoft Dataverse plugins. It allows step-through debugging, request capture, and execution tracing without deploying assemblies to a server environment. The tool is intended for development and diagnostic use only.
