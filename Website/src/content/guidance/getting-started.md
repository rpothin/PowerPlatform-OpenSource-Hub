## Welcome to the Power Platform Open Source Hub

The Power Platform Open Source Hub is a curated directory of open-source projects built for and around Microsoft's Power Platform ecosystem. Whether you're a seasoned developer or just getting started, there's a place for you here.

## What is Power Platform?

Microsoft Power Platform is a suite of low-code/no-code tools that includes:

- **Power Apps** — Build custom business applications without writing extensive code.
- **Power Automate** — Automate workflows and business processes across services.
- **Power BI** — Create interactive data visualizations and reports.
- **Copilot Studio** — Build intelligent conversational agents powered by AI.
- **Power Pages** — Create secure, data-driven external-facing websites.

## Finding Your First Project

1. **Browse the Discovery Grid** — Visit the [home page](/) and explore repositories sorted by popularity.
2. **Look for "Good First Issue" Labels** — Many projects tag beginner-friendly issues. Look for the 🤝 icon on repository cards.
3. **Read the Contributing Guide** — Each project has its own contributing guidelines. Always read them before submitting changes.
4. **Start Small** — Documentation fixes, typo corrections, and test additions are great first contributions.

## Understanding Repository Metrics

Each repository card shows key metrics:

- ⭐ **Stars** — A measure of community interest and approval.
- 🍴 **Forks** — How many developers have created their own copy to work on.
- 🤝 **Open to Contributions** — Number of issues marked as "good first issue" or "help wanted."
